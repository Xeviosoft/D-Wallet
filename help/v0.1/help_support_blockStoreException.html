BlockStoreException

You see a BlockStoreException very occasionally when MultiDoge starts. It prevents you synchronising with the Dogecoin network.

You can fix this problem by doing the following:

  1. Find where MultiDoge is storing your user data - see the help topic Troubleshooting

  2. Shut down MultiDoge

  3. Go to your MultiDoge user data directory - worked out in step 1 - and delete the file multidoge.spvchain

  4. Start up MultiDoge again

It will then regenerate this file and start syncing. After it has synced you should see all your dogecoin in your wallet..
