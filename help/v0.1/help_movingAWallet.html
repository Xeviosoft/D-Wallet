Moving a wallet to different directory

To move a wallet means copying several files at once. If you are not confident with using a computer like this then you should ask a trusted friend to help you.

You may want to move a wallet to a different location, such as on to a USB drive or a different directory. The files you will need to move will be named according to the original name you gave the wallet (not the description showing in the sidebar). For example, let's say you wanted to move the original wallet, "multidoge.wallet", created during installation:

  1. Ensure you have the destination available (e.g. USB stick plugged in, network drive mapped etc)
  2. Locate "multidoge.wallet" - the location appears in the title bar when you select the wallet in MultiDoge
  3. Multi-select "multidoge.wallet" (file) and "multidoge.info" (file) and "multidoge-data" (directory)
  4. Use your computer's Copy operation (not Move) to place the files into the destination
  5. Verify that they are OK by using MultiDoge's File | Open operation to access the wallet
  6. Close the original wallet to avoid confusion. Closing will not delete the wallet.
