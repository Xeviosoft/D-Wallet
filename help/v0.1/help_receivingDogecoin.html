Receiving dogecoin

To request dogecoin from someone else they need your "receiving address". You get your receiving address by choosing the " Request" tab or selecting the "Trade |  Request Dogecoin" menu option.

General procedure

  1. Create a new receiving address by clicking the "New" button
  2. Enter a label (e.g. "Payment for software development") to help you keep track of why you're receiving dogecoin
  3. Enter an amount value (e.g. 5.6789) in either the DOGE box or the local currency box

At this point MultiDoge will be able to show a QR code that a Dogecoin wallet on a mobile phone could use to send you dogecoin. The QR code contains the address, the label and the amount all in a convenient format.

Details

Dogecoin makes it easy to share your receiving address over the Internet. For example, copy your receiving address to the system clipboard by pressing the " Copy" button. Then paste the address directly into an email, Skype chat session etc to give it to the person sending you dogecoin.

You can also copy the generated QR code by pressing the " Copy" button in the right hand side panel. You can then send the QR code as an image to the payer by email, Skype etc. To help you with this if you drag the QR code directly to your desktop MultiDoge will turn it into a PNG image file automatically.

You can have as many receiving addresses as you like. MultiDoge recommends using a different receiving address for each person that is sending you dogecoin so that you can tell them apart. You create new receiving addresses by clicking on the "New" button on the " Request" tab.

In the "Label" field you should add a label to the receiving address to help you remember who is sending you dogecoin.
